Detail:

This comprehensive e-learning platform was developed using a stack of technologies including Django, Django Rest Framework (DRF), React, and Bootstrap. It caters to both learners and instructors, offering a seamless experience for course management, enrollment, and reviews.

Key Features:

User Authentication:

Users can create accounts, log in, and securely access their profiles. JWT token authentication ensures robust security.

Course Management:

Learners can explore a catalog of courses and purchase the ones they're interested in. Instructors can create and manage courses, adding content and details.

Course Reviews and Ratings:

Users can leave reviews and rate courses based on their experiences. Ratings contribute to course visibility and credibility.

Payment Integration:

React-paypal-js is used for seamless payment processing. Learners can securely pay for courses they wish to enroll in.

Frontend:

React, along with React Router DOM, provides dynamic and responsive user interfaces. Hooks like useEffect and useState enhance frontend functionality.

Backend:

Rest Framework (DRF) powers the backend. Class-based views, including ListAPIView, CreateAPIView and others handle course data retrieval.
